# countyer ++

A minimal, persistent counter web app. Increment or reset your tally, with instant updates and local storage persistence.

## Features
- Increment and reset counter
- Value persists across reloads (local storage)
- Responsive UI (desktop/mobile)
- No backend, no dependencies (Tailwind via CDN)

## Usage
1. Open `index.html` in your browser.
2. Click `+` to increment, `Reset` to reset.
3. Your count is saved automatically.

## Deploy
Host the folder on any static host (Netlify, Vercel, GitHub Pages, etc).
